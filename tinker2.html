Setelah mengikuti pembelajaran, murid diharapkan mampu:

  • ✅ Menggunakan antarmuka Tinkercad untuk membuat proyek Arduino secara virtual
  • ✅ Merancang rangkaian Arduino sederhana (LED blinking, sensor suhu, buzzer, dsb) menggunakan Tinkercad
  • ✅ Menulis dan menguji kode program Arduino dalam Tinkercad menggunakan bahasa C/C++
  • ✅ Melakukan simulasi dan analisis hasil kerja proyek Arduino di lingkungan Tinkercad
  • ✅ Memperbaiki (debugging) kesalahan dalam rangkaian atau program Arduino
  • ✅ Mempresentasikan hasil simulasi proyek Arduino dengan percaya diri

📚 Materi Pembelajaran

🌐 Apa itu Tinkercad?

Tinkercad adalah platform simulasi online gratis yang dikembangkan oleh Autodesk. Platform ini memungkinkan kita untuk:

  • ✅ Merancang rangkaian elektronik secara virtual
  • ✅ Mensimulasikan proyek Arduino tanpa hardware fisik
  • ✅ Belajar pemrograman Arduino dengan aman
  • ✅ Berbagi proyek dengan orang lain

💡 Keunggulan Tinkercad

Gratis • Mudah digunakan • Tidak perlu instalasi • Simulasi real-time • Komunitas besar

📝 Tugas & Aktivitas Lanjutan

🎯 Tugas Individu

📋 Tugas 1: Proyek LED Blinking

Instruksi:

  • Buat akun Tinkercad di www.tinkercad.com
  • Buat rangkaian LED yang berkedip setiap 1 detik
  • Gunakan Arduino Uno, LED, dan resistor 220Ω
  • Tulis program untuk mengontrol LED
  • Screenshot hasil simulasi dan kirim ke guru

Waktu: 1 minggu

🔔 Tugas 2: Sistem Alarm Sederhana

Instruksi:

  • Buat rangkaian dengan buzzer dan LED
  • Program agar buzzer berbunyi dan LED berkedip bersamaan
  • Tambahkan tombol untuk mengaktifkan/menonaktifkan alarm
  • Dokumentasikan proses pembuatan dalam video singkat (2-3 menit)

Waktu: 1 minggu

👥 Tugas Kelompok: Proyek Kreatif

Instruksi:

  • Bentuk kelompok 3-4 orang
  • Pilih salah satu: Lampu lalu lintas, Sensor suhu dengan display, atau Sistem keamanan pintu
  • Buat presentasi 10 menit tentang proyek kalian
  • Sertakan: Diagram rangkaian, kode program, dan demo simulasi

Waktu: 2 minggu
